# ImageConvert React Migration

This repository has been migrated from static HTML pages to a React + Vite application.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Migration scope

- React Router based routing for home, tools, blog and info pages.
- Auto-generated data catalogs for all tools/blog/info pages from legacy HTML slugs.
- Reusable templates for tool details and content pages.
